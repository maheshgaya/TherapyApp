# CS188Project3
CS188 Project 3 Group 1

## Using the Source Code
- Clone the project 
```
  $ git clone https://github.com/maheshgaya/CS188Project3.git
  $ cd CS188Project3/
```

- Import the project in Android Studio

```
  File -> New -> Import Project -> (Choose CS188Project3/MMELK/)
```
